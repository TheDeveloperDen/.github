## Getting Started with Contributing

Want to contribute but not sure where to start? Pick a project that fits your current skills!
Once you've picked a project, check out its **Issues** page to find useful ways to contribute!

### ðŸŸ¢ Level 1: "I'm new to open source"

These projects are designed to be accessible for new contributors and don't require any complex setup

| Project                                                                       | Tech Stack                | Description                                                                                                                       |
| :---------------------------------------------------------------------------- | :------------------------ | :-------------------------------------------------------------------------------------------------------------------------------- |
| [**LearningResources**](https://github.com/TheDeveloperDen/LearningResources) | YAML + Markdown           | A crowdsourced list of useful programming tutorials and resources. Add a link to your favorite guide and help someone else learn! |
| [**DevBin (Frontend)**](https://github.com/TheDeveloperDen/DevBin)            | Svelte + TypeScript + CSS | Our pastebin's web UI                                                                                                             |

### ðŸŸ  Level 2: "I have some coding experience"

Bigger projects with more complex setups (e.g. Docker)

| Project                                                                                 | Tech Stack   | Description                                                                 |
| :-------------------------------------------------------------------------------------- | :----------- | :-------------------------------------------------------------------------- |
| [**DevBin (Backend)**](https://github.com/TheDeveloperDen/DevBin)                       | Python + SQL | The backend API powering our pastebin                                       |
| [**discord-bot**](https://github.com/TheDeveloperDen/discord-bot)                       | TypeScript   | Our main Discord Bot, which handles XP, moderation, and more                |
| [**Codosseum Backend**](https://github.com/codosseum-org/backend)                       | Java         | The main backend service for **Codosseum**, our WIP competitive coding game |
| [**Codosseum Challenges Service**](https://github.com/codosseum-org/challenges-service) | Kotlin       | Microservice handling challenge data for **Codosseum**                      |

### Suggestions

Don't see anything you like and have a cool idea? **Suggest something!** We're open to all ideas, so let us know on Discord or make an Issue in this repository.
